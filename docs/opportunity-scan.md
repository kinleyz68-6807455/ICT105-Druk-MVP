# Lab 02 Opportunity Scanning
## Ideas Reviewed from Lab 1
| Idea | Problem Area | Target User | Current Alternative | Initial Technology Direction |
|---|---|---|---|---|
| Idea 1: Tourism Information Platform | Travelers struggle to find reliable and organized tourism information in one place. | Tourists, local travelers, students, backpackers, travel planners | Google searches, travel blogs, social media, booking websites | HTML, CSS, JavaScript, responsive web design, cloud hosting, optional Google Maps API |
| Idea 2: Smart Campus Event Hub | Students miss campus events because information is scattered across many channels. | University students, clubs, campus administrators | Posters, messaging apps, social media groups | Web or mobile app, notification system, cloud database |
| Idea 3: Local Artisan Marketplace | Local artisans have limited online visibility and customer reach. | Artisans, small business owners, customers | Physical shops, local markets, social media pages | Web application, product catalog system, cloud hosting and storage |
